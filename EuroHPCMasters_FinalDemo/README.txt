A slurm script named slurm.sh is provided.
You can submit this script like sbatch slurm.sh
Please make sure you change the account name and qos name in the script. 
You can play with the number of processes (try not to give a prime number). 
Before executing make sure you read the problem description document properly. 
If the program executes successfully (fingers crossed), it will output
some .dat files and a solution.pdf and and solution.png. 
You can copy these PDF/PNG files to your local laptop/computer to see how the solution changes.
